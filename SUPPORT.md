# Support

## Questions

Use GitHub Discussions if they are enabled for the repository. Otherwise, open a GitHub issue with the `question` label.

## Bugs

Open a bug report and include:

- Operating system and version.
- codexU version.
- Whether Codex is installed and signed in.
- Minimal reproduction steps.

Remove private Codex thread titles, local paths, account data, token data, and screenshots with private work before posting.

## Security

Do not open public issues for security-sensitive reports. Follow [SECURITY.md](SECURITY.md).
