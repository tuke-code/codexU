# codexU v1.0.2

本次版本重点优化菜单栏信息定制与可读性，让不同屏幕空间和使用习惯都能选择合适的展示密度。

## 主要更新

- 新增简约、经典、丰富三档状态栏展示模式。
- 支持在“已用量 / 剩余量”之间切换，并用相反的进度方向表达口径。
- 支持选择 5 小时额度、7 天额度、今日 token 与重置倒计时。
- 简约模式使用无 Logo 的加粗蓝紫双环；经典模式使用纯数字额度环；丰富模式保留完整标签、进度条和重置时间。
- 状态栏背景保持透明，品牌 Logo 使用从原始图形派生的系统单色模板，并随菜单栏实际深浅自动适配。
- 提高辅助标签和重置时间的对比度，今日总量使用系统菜单栏正文尺寸。
- 设置窗口新增即时预览，所有显示配置即时保存并应用。

## 安装包

- Intel：`codexU-1.0.2-mac-x86_64.dmg`
